# Messi - Professional Portfolio Website

A complete professional portfolio website with an integrated admin panel built with React, Express, and SQLite.

## Features

### Public Website
- **Home**: Hero section with professional introduction
- **Projects**: Dynamic project showcase fetched from database
- **Services**: Web/App Development, Bot Development, Automation Tools, Ethical Hacking
- **Skills**: AI, Machine Learning, Python, Node.js, and more
- **Contact**: Contact form with social links

### Admin Panel
- Secure login with hardcoded credentials
- Projects Management (CRUD operations)
- Contact Messages viewer with delete functionality
- Image upload for project previews

## Tech Stack

- **Frontend**: React, Tailwind CSS, Shadcn UI, Framer Motion
- **Backend**: Express.js, Node.js
- **Database**: SQLite (local file-based)
- **Authentication**: Session-based with HTTP-only cookies
